Pinceau

Il suffit de séléctionner le pinceau dans la barre de gauche. Une fois séléctionné, il est possible de cacher la section 'attributs d'outils' en recliquant sur l'outil séléctionné. La section attributs outils permet d'ajuster l'épaisseur du Pinceau et le menu déroulant 'Options de couleur' permet de modifier la couleur sélectionnée. Il est également possible de choisir une texture de pinceau parmis les 5 options disponibles.
