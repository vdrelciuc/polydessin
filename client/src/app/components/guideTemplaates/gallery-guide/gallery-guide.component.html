Gallerie

Il est possible de récupérer les images stockées sur le serveur dans la gallerie. Par la suite, il est possible de les filter en ajoutant des étiquettes. Finalement, pour supprimer ou charger un dessin , il faut se positioner sur la prévisualisation de celui-ci puis cliquer sur le bouton désiré. Le raccourci 'ctrl G' ouvre la gallerie.
