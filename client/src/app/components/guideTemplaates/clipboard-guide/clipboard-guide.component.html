Presse-Papier

Pour utiliser le presse papier, il est nécessaire de vérifier la présence sur le menu de selection. Suite à cela, il est possible de manipuler les éléments du dessin avec les boutons présents dans le menu attributs d'outils ou bien avec les combinaisons suivantes : "ctrl+X : couper ; ctrl+c : copier ; ctrl+v : coller ; ctrl+d : duppliquer". Une séléction peut par ailleurs être effacé. Il suffirait de se placer dessus puis de peser sur la touche "delete".
