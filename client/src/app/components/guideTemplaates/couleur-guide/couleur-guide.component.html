Couleur

Il suffit de séléctionner le menu 'Options de Couleur' dans la barre d'attributs. Les 10 dernières couleurs utilisées sont présentées et prêtes à être utilisées.

Il est possible de changer la couleur d'arrière plan en appuyant sur le bouton " Changer l'arrière plan". Par la suite une pallette de couleur s'affiche.

Sur cette Palette, il est possible de selectionner la couleur de base souhaitée dans le slider. Suite à ceci, il faut choisir le dégradé voulu dans le grand carré coloré. Il est aussi possible d'entrer une valeur manuel en hexadécimale dans les champs 'R' 'V' 'B'.
